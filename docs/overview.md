# Godot Audio Sequencer Tool — Overview

## Purpose

Godot Audio Sequencer Tool is a standalone Godot editor plugin for building and editing sequenced audio timelines inside the Godot editor.

The goal is to provide a reusable music sequencing tool that can be integrated into different projects without being tied to one specific game.

## Current direction

The tool is being designed as a bottom-panel editor plugin with:
- a timeline view
- multiple tracks
- clip-based audio arrangement
- lightweight saved song data
- support for in-editor composition

## Core goals

- reusable across projects
- editor-integrated workflow
- timeline-based audio editing
- support for layered and synchronized audio
- lightweight project-side save format
- fluid usability to avoid having to use dedicated software for making simple audio
